Adopt Me



Este game pode soar levemente estranho (afinal, você deve escolher se quer ser um pai ou um bebê).
Caso escolha a primeira opção, deve seguir para uma área onde outros jogadores que estão como bebê farão de tudo para serem adotados.
Porém, a jornada não para por aí. Para o pai, é preciso se preocupar em manter as barras de Fome, Diversão, Limpeza e Sono balanceadas, enquanto
o bebê tem acesso a vários minigames que certamente vão deixar o cuidador maluquinho.
